Live Writer에 좋아하다가 Word에서도 블로깅이 된다고 하더라.
어느 정도 억울하다. 뭔가 허전하고 Live Writer, 별로 무겁지도 않은 프로그램이지만 설치하는 데에는 꽤 시간이 걸린다.

Word에서 블로그를 작성하는 것, 그건 꽤나 해보고 싶은 짓이다.
본인의 Tistory 계정에서의 블로그에 한 번 블로깅을 해보겠다.

해본 적이 없어서 결과는 어떻게 나올지 모르지만…
(결과는 댓글에다 써넣어야지.)

## 1. Word에서 블로깅하는 방법 찾기

1. Office 단추 클릭 (왼쪽 상단)

![office button](http://moneyup.host6.hostment.info/attach/1/ck200000000000.png)

2. **새로 만들기 → 새 블로그 게시물**

## 2. 계정 설정

(최초 게시할 때 자동으로 설정 창에 들어감)

1. 새 계정 → 기타
2. API: **MetaWeblog**로 설정 (ATOM 아님)
3. 블로그 URL: `계정.tistory.com/api`
   - 위치: 블로그 관리자 → 환경설정 → 기타설정 하단
4. 사용자명: tistory ID
5. 비밀번호: tistory 비밀번호

이러고 열심히 쓰면 됩니다.
아직 게시도 안 했는데, 이제 게시를 해봐야겠군요.

## 3. 초안으로 게시하기

![draft](http://moneyup.host6.hostment.info/attach/1/ck200000000001.png)

왼쪽 상단의 **게시 버튼**

게시와 **초안으로 게시**가 있습니다.

Live Writer와 달리 초안으로 게시는
컴퓨터에 저장되는 것이 아니라 **계정의 비공개 글로 저장**됩니다.

## 4. 게시하기

게시를 하면 그냥 올라갑니다.

태그 설정은 안 되는 것 같고
분류(그림의 범주 삽입)는 설정할 수 있습니다.

트랙백과 답변에 대한 설정은 할 수 없습니다.
